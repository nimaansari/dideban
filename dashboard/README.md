# دیده‌بان | Dideban

داشبورد زنده اخبار نظامی و اطلاعاتی خاورمیانه

## ویژگی‌ها

- 📡 فید زنده اخبار تلگرام و توییتر
- 🗺 نقشه تعاملی رویدادهای نظامی
- 💥 حملات WarStrikes و INSS روی نقشه
- 🔴 تشخیص خودکار اخبار فوری و رسمی
- 🌐 ترجمه خودکار متون انگلیسی به فارسی
- ⚡ آپدیت زنده بدون نیاز به رفرش

## اجرا

```bash
pip install -r requirements.txt
uvicorn server:app --host 0.0.0.0 --port 8080
```

## منابع داده

- کانال‌های تلگرام فارسی
- X / Twitter
- WarStrikes API
- INSS ArcGIS Data
